nand2tetris, or How to Build a Computer from First Principles

02 Aug 2015

Last month (almost) I finished one of the most interesting and enjoyable Computer Courses on Coursera, the nand2tetris: Part I Course, or Build a Modern Computer from First Principles: Nand to Tetris.

The MOOC is based on the book (“Elements of Computing System”)[https://mitpress.mit.edu/books/elements-computing-systems], and the self-paced course that uses the book which is outlined on the website http://www.nand2tetris.org/.

In a nutshell, what you do is basically start with a language that looks like a minified version of Verilog or VHDL called HDL (Hardware Description Language) and a Nand gate implementation. From there, you implement all elementary single-bit logic gates:

  • And
  • Or
  • Not
  • Xor
  • Mux: multiplexer
  • DMux: demultiplexer
  • DMux4Way: 4-ways 1-bit demultiplexer
  • DMux8Way: 8-ways 1-bit demultiplexer

And some other multi-bits gates:

  • And16: 16-bits AND gate
  • Or16: 16-bits OR gate
  • Mux16: 16-bits MUX gate
  • Mux4Way16: 4-ways 16-bit multiplexer
  • Mux8Way16: 8-ways 16-bit multiplexer
